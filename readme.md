# Euclid

[Euclid](https://euclid.tulv.in/) is a minimal flip clock application. Unlike other flip clocks, Euclid iteratively changes the digits by flipping only the necessary segments, resulting in less visual noise. I primarily built this to be run as the Hub Mode clock of a Pixel Tablet.
